# 🔍 Network Port Scanner (Python + Tkinter)

A fast and user-friendly **multi-threaded port scanner** built using Python with a GUI interface. This tool scans a target IP/hostname to identify open ports and associated services.

---

## 🚀 Features

* ⚡ Fast scanning using **ThreadPoolExecutor**
* 🖥️ Simple and clean **Tkinter GUI**
* 🔍 Scans custom **port ranges**
* 📡 Detects **common services (HTTP, SSH, FTP, etc.)**
* ⏹️ Start / Stop scan functionality
* 📊 Real-time progress updates
* 💾 Save results to a file

---

## 🛠️ Tech Stack

* Python 3
* Tkinter (GUI)
* Socket Programming
* Multithreading (ThreadPoolExecutor)

### 1. Clone the repository

```bash

```

### 2. Run the application

```bash

```

## 🧪 Usage

1. Enter target IP or hostname
2. Enter port range (e.g., 1–1024)
3. Click **Start Scan**
4. View open ports in real-time
5. Save results if needed
